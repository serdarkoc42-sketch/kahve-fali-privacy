# kahve-fali-privacy
# Kahve Falı - Gizlilik Politikası

**Son güncelleme:** 17 Mayıs 2026

## 1. Genel Bakış

Kahve Falı uygulaması ("Uygulama"), yapay zeka destekli kahve falı yorumu sunan bir eğlence uygulamasıdır. Bu gizlilik politikası, uygulamanın hangi verileri topladığını, nasıl kullandığını ve koruduğunu açıklamaktadır.

## 2. Toplanan Veriler

### 2.1 Fotoğraflar
- Kullanıcı tarafından yüklenen kahve fincanı fotoğrafları yalnızca yapay zeka analizi için kullanılır.
- Fotoğraflar sunucularımızda **kalıcı olarak saklanmaz**.
- Analiz tamamlandıktan sonra fotoğraflar otomatik olarak silinir.
- Fotoğraflar üçüncü taraflarla paylaşılmaz.

### 2.2 Kullanım Verileri
- Uygulama içi günlük kullanım sayacı (yerel cihaz belleğinde tutulur).
- Premium üyelik durumu (yerel cihaz belleğinde tutulur).
- Bu veriler yalnızca cihazınızda saklanır, sunucuya gönderilmez.

### 2.3 Reklam Verileri
- Uygulama, Google AdMob reklam hizmeti kullanmaktadır.
- AdMob, reklam kişiselleştirme amacıyla cihaz tanımlayıcıları toplayabilir.
- Google'ın gizlilik politikası için: https://policies.google.com/privacy

### 2.4 Yapay Zeka Analizi
- Fotoğraflar Google Gemini API aracılığıyla analiz edilir.
- Analiz için fotoğraflar şifreli bağlantı (HTTPS) üzerinden iletilir.
- Google'ın veri işleme politikası için: https://ai.google.dev/terms

## 3. Veri Güvenliği

- Tüm veri iletimi HTTPS ile şifrelenir.
- API anahtarları uygulama kodunda bulunmaz, güvenli sunucu ortam değişkenlerinde saklanır.
- Fotoğraflar sunucuda işlendikten sonra silinir.

## 4. Üçüncü Taraf Hizmetler

Uygulama aşağıdaki üçüncü taraf hizmetleri kullanmaktadır:

| Hizmet | Amaç | Gizlilik Politikası |
|--------|-------|---------------------|
| Google Gemini API | Kahve falı analizi | https://ai.google.dev/terms |
| Google AdMob | Reklam gösterimi | https://policies.google.com/privacy |
| Google Cloud Run | API proxy hizmeti | https://cloud.google.com/privacy |

## 5. Çocukların Gizliliği

Bu uygulama 13 yaşın altındaki çocuklara yönelik değildir. Bilerek 13 yaşın altındaki çocuklardan kişisel veri toplamıyoruz.

## 6. Eğlence Amaçlı Kullanım

Bu uygulama **yalnızca eğlence amaçlıdır**. Sunulan kahve falı yorumları:
- Kesin gelecek tahmini niteliği taşımaz.
- Profesyonel tavsiye yerine geçmez.
- Bilimsel bir temele dayanmaz.

## 7. Kullanıcı Hakları

Kullanıcılar aşağıdaki haklara sahiptir:
- Uygulama tarafından işlenen veriler hakkında bilgi talep etme.
- Yerel cihaz verilerini uygulama ayarlarından silme.
- Uygulamayı kaldırarak tüm yerel verileri silme.

## 8. Gizlilik Politikası Değişiklikleri

Bu politikayı zaman zaman güncelleyebiliriz. Önemli değişiklikler olması durumunda uygulama içinde bildirim yapılacaktır. Güncel politika her zaman bu sayfada yayımlanacaktır.
